# git-practice-sage-marigold

This repository is a "safe space" to practice the Git workflow we will be using for our hackathon project. Don't worry about breaking anything that's what this repo is for!

## Your Mission:
Your goal is to add a file with your name/discord username to the `profiles/` folder. you can add hobbies or quotes for file content. make sure to save the file with [your-username].txt 

### Step-by-Step Instructions

1. **Fork the Repo**
   - Click the **Fork** button at the top right of this page. This creates a copy in YOUR GitHub account.

2. **Clone Your Fork**
   - Open your terminal and run:
     `git clone https://github.com/YOUR-USERNAME/git-practice-sage-marigold.git`
   - `cd git-practice-sage-marigold`

3. **Create a New Branch**
   - `git checkout -b profile-add-[your-username]`

4. **Add Your File**
   - Create a new file in the `profiles/` folder named `your-discord-username.md`.
   - Inside the file, write a quick "Hello!" or what you're excited to build.

5. **Save and Push**
   - `git add .`
   - `git commit -m "Added my profile to the wall of fame"`
   - `git push origin profile-add-[your-username]`

6. **Open a Pull Request**
   - Go to the **original** repo on GitHub.
   - You'll see a banner click **Compare & pull request**.
   - Submit it! I'll review and merge it.
